# Lembar Observasi Usability Testing

## Task yang Diuji
1. Register akun
2. Login ke sistem
3. Mencari lapangan
4. Menggunakan filter lapangan
5. Melihat detail lapangan
6. Melihat jadwal lapangan
7. Melakukan booking lapangan
8. Melakukan pembayaran
9. Melihat riwayat booking

## Hasil Observasi

| No | Aktivitas | Hasil Observasi |
|---|---|---|
| 1 | Register akun | Pengguna berhasil membuat akun |
| 2 | Login sistem | Pengguna berhasil login tanpa kendala |
| 3 | Cari lapangan | Pengguna dapat menemukan lapangan dengan mudah |
| 4 | Filter lapangan | Filter membantu pengguna memilih lapangan |
| 5 | Detail lapangan | Pengguna membaca informasi lapangan sebelum booking |
| 6 | Jadwal lapangan | Pengguna sempat bingung memilih slot jadwal |
| 7 | Booking lapangan | Pengguna berhasil melakukan booking |
| 8 | Pembayaran | Pengguna memahami proses pembayaran |
| 9 | Riwayat booking | Pengguna dapat melihat riwayat booking |

## Catatan Observer

- Pengguna memahami alur booking dengan baik.
- Pengguna sempat bingung dalam memilih jadwal lapangan.
- Pengguna berhasil menyelesaikan proses booking setelah diarahkan.
- Tampilan sistem dinilai cukup nyaman dan mudah dipahami.
