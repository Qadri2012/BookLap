# SUS Questionnaire (System Usability Scale)

Berikan penilaian dari skala 1–5:

1 = Sangat Tidak Setuju  
2 = Tidak Setuju  
3 = Netral  
4 = Setuju  
5 = Sangat Setuju

---

1. Saya merasa sistem mudah digunakan.
2. Saya merasa sistem ini sederhana dipahami.
3. Saya merasa fitur-fitur pada sistem terhubung dengan baik.
4. Saya percaya diri menggunakan sistem ini.
5. Saya merasa alur booking cukup jelas.
6. Saya merasa tampilan sistem nyaman dilihat.
7. Saya ingin menggunakan sistem ini lagi.
8. Saya merasa sistem ini cepat dipelajari.
9. Saya tidak menemukan kesulitan berarti saat memakai sistem.
10. Saya puas dengan pengalaman menggunakan sistem ini.
